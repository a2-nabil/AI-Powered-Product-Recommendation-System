AI-Powered Product Recommendation System enhances user experience by providing personalized product recommendations using Google Gemini AI.

== Description ==
AI-Powered Product Recommendation System is an intelligent WordPress plugin that enhances user experience by providing personalized product recommendations. Using Google Gemini AI, this chatbot-powered system analyzes user preferences and interactions to suggest the most relevant products dynamically.

Features:

AI-driven personalized product suggestions.

Seamless integration with WordPress WooCommerce stores.

Chatbot functionality to enhance user interaction.

Uses Google Gemini AI for intelligent recommendations.

Customizable settings for product filtering and display.

Lightweight and optimized for fast performance.

== Installation ==

Upload the plugin files to the /wp-content/plugins/ai-powered-product-recommendation-system directory, or install the plugin through the WordPress plugins screen directly.

Activate the plugin through the 'Plugins' screen in WordPress.

Configure API settings for Google Gemini AI in the plugin settings.

Customize product recommendation filters based on user preferences.

== Frequently Asked Questions ==
= How does this plugin recommend products? =
The plugin utilizes Google Gemini AI to analyze user input and preferences, providing real-time personalized recommendations.

= Does this plugin require WooCommerce? =
Yes, WooCommerce is required to fetch and recommend products from your store.

= Can I customize the recommendation settings? =
Yes, you can define product filters, weightage, and display preferences in the plugin settings.
